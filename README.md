# Issue Creation Testing

This repository contains tests for the automatic issue creation feature.

## Prerequisites

- Python 3.8+
- pytest
- Your project's issue creation module

## Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Update `test_issue_creation.py` with your actual issue creation module path
2. Run tests:
   ```bash
   pytest
   ```

## Tests Included

- Valid issue creation
- Issue creation with missing required fields
- Issue creation with invalid parameters
- Issue creation with external dependency failures
